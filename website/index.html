Go HTTP Router

Build clean APIs fast with typed handlers and zero ceremony.

`box` gives you an ergonomic router with route groups, composable interceptors, automatic JSON flow, OpenAPI generation and first-class WebSocket routes.

Read Documentation Go Reference 
go get github.com/fulldump/box

Typed by default

Use Go structs directly in handlers for request and response payloads.

Composable middleware

Attach interceptors globally, per group, or per route with predictable ordering.

WebSocket ready

Create upgrade routes with `HandleWebSocket` and custom upgrade policies.

OpenAPI support

Generate and publish API specs directly from the routes you define.

Quick Example

package main

import (
  "net/http"
  "github.com/fulldump/box"
)

func main() {
  b := box.NewBox()
  b.Handle(http.MethodGet, "/hello/{name}", func(r *http.Request) map[string]string {
    return map[string]string{"message": "hello " + r.PathValue("name")}
  })
  _ = b.ListenAndServe()
}
